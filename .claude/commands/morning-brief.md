# /morning-brief — Daily Briefing

Generate a morning briefing to start the day focused.

1. Read context files for current priorities
2. Check for any recent changes in the workspace
3. Identify:
   - Top 2-3 priorities for today
   - Active tasks and their status
   - Upcoming deadlines
   - Quick wins that can be cleared fast
4. Suggest a focus order for the day

Keep it to one screen — scannable in 60 seconds.
