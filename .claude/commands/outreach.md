# /outreach — Draft Outreach

Draft personalized outreach emails based on prospect research.

1. Research the target person/company (if web access available)
2. Find a personalized angle (recent news, shared connection, relevant challenge)
3. Draft a short, compelling email:
   - Personalized opening
   - One specific value proposition
   - Social proof
   - Low-friction CTA
4. Provide 3 subject line options
5. If requested, generate a 3-4 email follow-up sequence

Usage: `/outreach [person or company name] [optional context]`
Examples:
- `/outreach Jane Smith at Acme Corp — interested in our platform`
- `/outreach sequence for enterprise prospects in automotive`
