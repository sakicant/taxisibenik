# /report — Generate Report

Generate a structured report from available data and context.

1. Read relevant context files, data, and any specified sources
2. Analyze the data for trends, patterns, and insights
3. Structure the report:
   - **Summary** — Key metrics and headline findings
   - **Performance** — Detailed breakdown by relevant segments
   - **Trends** — What's improving, declining, or stable
   - **Insights** — What the data tells us
   - **Recommendations** — Actions to take based on findings
4. Include comparison periods where data is available
5. Save to outputs/ if requested

Usage: `/report [type and scope]`
Examples:
- `/report monthly marketing performance`
- `/report pipeline review for Q2`
- `/report event ROI for Tech.AD Europe`
