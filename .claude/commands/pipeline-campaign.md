# /pipeline-campaign: Campaign Launch Pipeline

Run the full campaign launch pipeline from planning through to all assets.

## Steps

### 1. Campaign plan
Define objectives, audience, channels, timeline, budget, and success metrics.

Output: campaign brief document.

### 2. Landing page copy
Write landing page copy: hero section, value props, social proof, CTA. Follow brand voice guidelines.

Output: landing page copy document.

### 3. Email sequence
Write a launch email sequence: teaser, announcement, follow-up, last chance. Match the audience and messaging from the campaign plan.

Output: 4 email drafts.

### 4. Social calendar
Create a 2-week social posting schedule. Coordinate timing with the email sequence.

Output: social calendar with copy for each post.

## How to use

Run: `/pipeline-campaign [campaign name or brief]`

Present output at each stage for approval before continuing.
