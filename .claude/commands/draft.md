# /draft — Quick Content Draft

Draft content based on a brief description.

1. Identify the content type (blog, email, social post, ad copy, landing page, etc.)
2. Check context files and reference files for brand voice and guidelines
3. Ask clarifying questions only if critical information is missing
4. Write a first draft following the appropriate structure for the content type
5. Present with formatting ready to copy

Usage: `/draft [description]`
Examples:
- `/draft blog post about our new feature`
- `/draft LinkedIn post announcing the partnership`
- `/draft follow-up email for the demo we gave yesterday`
