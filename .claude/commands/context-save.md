# /context-save — Save Session Context

Capture the current session's context for future reference.

1. Summarize what we've been working on
2. Document any decisions made and their rationale
3. Note any work in progress or next steps
4. Save to `context/session-notes.md` (append, don't overwrite)
5. Update CLAUDE.md if any structural changes were made

This helps future sessions pick up where we left off.
