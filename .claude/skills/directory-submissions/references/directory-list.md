# Directory List

Tier 1 — high-DR, slow review, worth the wait:

- Product Hunt — productHunt.com — daily launches, voting-based.
- BetaList — betalist.com — pre-launch products, early-adopter audience.
- AlternativeTo — alternativeto.net — when users search "[competitor] alternative".
- G2 — g2.com — B2B reviews, expensive but high-intent.
- Capterra — capterra.com — same audience as G2, often co-listed.
- SaaSHub — saashub.com — broad SaaS directory.
- Crunchbase — crunchbase.com — company profile, signals to investors and partners.

Tier 2 — fast review, cumulatively meaningful:

- TAAFT (There's An AI For That) — theresanaiforthat.com — large AI tool index.
- Futurepedia — futurepedia.io — AI tool catalog.
- aitools.fyi — aitools.fyi — AI tool listings.
- Futuretools — futuretools.io — AI tool catalog.
- ToolsPond — broader productivity tools.
- TopAITools — topaitools.com — featured AI tools.

Tier 3 — niche / cumulative:

- AI agent / MCP registries (search for current ones — landscape shifts monthly).
- Dev tool directories (DevHunt, Hacker News Show HN, Reddit r/SideProject).
- No-code directories (NoCodeList, MakerLog).
- Newsletter directories (DiscoverEmails, etc.).
- Country/region-specific (yes-startup directories in your region).

Always-skip:

- Pay-to-list directories with no editorial filter (they hurt more than help).
- Directories with broken UX or dead links (signal of low traffic).
- Directories blocked by major search engines.

Update this list quarterly — the directory landscape changes.
